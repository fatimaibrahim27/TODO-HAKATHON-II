# Frontend Skill

## Instructions

1. **UI Generation**
   - Create pages and components using Next.js App Router
   - Build responsive layouts that work on all devices
   - Follow design specs from UX/UI requirements

2. **Performance Optimization**
   - Detect and fix performance bottlenecks
   - Reduce unnecessary re-renders and computations
   - Optimize asset loading and bundle size
   - Apply lazy loading where appropriate

3. **Component & State Management**
   - Keep components modular and reusable
   - Use client components only when interactivity is needed
   - Manage state efficiently for fast UI updates

4. **Integration**
   - Connect frontend components with backend API endpoints
   - Include authentication headers (JWT) in API calls
   - Handle API responses and display errors gracefully

## Best Practices
- Mobile-first approach
- Use Tailwind CSS classes, avoid inline styles
- Follow consistent component patterns
- Optimize images and static assets

## When to Use
- Creating new pages or UI components
- Optimizing rendering and load performance
- Connecting frontend to backend APIs
- Implementing authentication flows in UI
